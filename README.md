# Sismógrafo – Estimación de Riesgo Sísmico con ML

Desarrollo de un prototipo de plataforma digital que estima el riesgo sísmico
en Oaxaca mediante técnicas de Machine Learning.

## Objetivo
Desarrollar un sistema que integre bases de datos, modelos de ML y una interfaz
para clasificar niveles de riesgo sísmico y simular escenarios de afectación.

## Tecnologías
- Python (Flask, Pandas, NumPy, PyTorch)
- PostgreSQL
- HTML, CSS, JavaScript
- Git & GitHub

## Alcance
Este sistema no predice sismos. Realiza estimaciones de riesgo basadas en datos históricos.
